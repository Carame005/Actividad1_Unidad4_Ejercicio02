# Capturas de Pantalla

## Instrucciones

Coloca en esta carpeta las siguientes capturas:

### 1. programa_enviado.png
Captura del programa mostrando:
- ✅ Configuración SMTP completa (host, puerto, username)
- ✅ Datos del correo (From, To, Subject)
- ✅ Contenido HTML en la ventana izquierda
- ✅ Log en la ventana derecha mostrando:
  - "Autenticación exitosa"
  - "¡Correo enviado exitosamente!"
  - "Verifique su inbox en Mailtrap"

### 2. mailtrap_recibido.png
Captura del inbox de Mailtrap mostrando:
- ✅ El correo recibido en la lista de mensajes
- ✅ Detalles del correo (From, To, Subject)
- ✅ Contenido HTML renderizado correctamente
- ✅ Fecha y hora de recepción

## Cómo Tomar las Capturas

### Windows
- **Captura completa**: Tecla `PrtScn` (Imprimir Pantalla)
- **Captura de ventana activa**: `Alt + PrtScn`
- **Herramienta de recortes**: `Win + Shift + S`

### Guardar las Capturas
1. Pegar en Paint o cualquier editor de imágenes
2. Guardar con los nombres especificados arriba
3. Colocar en esta carpeta `/capturas/`
